# Visit My Page - Personal Portfolio

A modern, high-tech React portfolio with a secure Admin Panel for managing contact submissions.

## 🚀 Key Features
- **Modern Design**: Futuristic UI with smooth animations and a responsive layout.
- **Admin Panel**: Secure portal to monitor and manage contact transmissions.
- **Contact Sync**: Integrated with MongoDB to save messages while maintaining WhatsApp redirect.
- **Automated Deploy**: GitHub Actions automatically updates the live site on every push.

---

## 🛠️ Installation & Setup

### 1. Prerequisites
- **Node.js** (v18+)
- **MongoDB** (Running locally on port 27017 or a cloud instance)

### 2. Backend Setup
```bash
cd backend
npm install
# Create a .env file (use .env.example as a template)
node server.js
```

### 3. Frontend Setup
```bash
npm install
npm run dev
```
